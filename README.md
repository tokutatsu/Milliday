# Milliday

## 概要
twitterでミリシタの誕生日を呟くBotです。   
一応汎用性を上げようとしたので、./data/birthday.jsonのデータを変更すれば対応可能です。

インストール
```
npm install
```

token.jsonを作成して、使用したいBotのトークンをjson形式で記述してください。
```
{
  "consumer_key": "***",
  "consumer_secret": "***",
  "access_token_key" "***"
  "access_token_secret" "***"
}
```
