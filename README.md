# Milliday

## 概要
twitterでミリマスのキャラの誕生日を呟くBotです。   
一応汎用性を上げようとしたので、./data/birthday.jsonのデータを変更すれば対応可能となっています。

## モジュールのインストール
```
npm install
```

## Botのトークンファイル作成

token.jsonを作成して、以下のように使用したいBotのトークンをjson形式で記述してください。
```
{
  "consumer_key": "***",
  "consumer_secret": "***",
  "access_token_key" "***"
  "access_token_secret" "***"
}
```
